## Get started

<Steps>
  <Step title="Open Neroli">
    Launch Neroli on your Mac and choose a repository.
  </Step>

  <Step title="Create a session">
    Start a new agent session from the repository workspace.
  </Step>

  <Step title="Review the work">
    Inspect the files, terminal output, and changes from the session before merging anything back.
  </Step>
</Steps>

<Tip>
  These docs are intentionally minimal while the product documentation is being built out.
</Tip>
