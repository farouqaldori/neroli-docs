Neroli is a native Mac app for coordinating AI coding agents across isolated Git worktrees.

Use Neroli when you want to run agent work in parallel, see what each session changed, keep terminals and browser panes attached to the right worktree, and review the result before merging.

## What Neroli helps with

- Running Claude, Codex, and shell-based workflows in isolated worktrees.
- Keeping parallel tasks separate until you decide what to merge.
- Watching files, terminal output, browser state, simulator state, and diffs in one place.
- Capturing repeatable project actions as `.neroli/automations`.
- Moving supported sessions between local and cloud runtime when cloud is enabled.

<Info>
  Neroli isolates development state. It is not a sandbox boundary. Local commands and agents still run with your macOS user permissions.
</Info>
