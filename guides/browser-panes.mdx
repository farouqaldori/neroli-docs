---
title: "Use browser panes"
description: "Attach web verification to the right Neroli session."
---

Browser panes let you inspect web apps from the same session that owns the code changes. They are useful for visual checks, DOM inspection, and reproducing user flows.

## Open a page

Use the app UI or the CLI:

```sh
neroli browser open http://localhost:4321 --name website
neroli browser look --browser website
```

## Inspect before interacting

```sh
neroli browser look
neroli browser state --role button --name-regex Save
neroli browser read body --max-chars 2000
```

## Click and verify

```sh
neroli browser tap "Save"
neroli browser wait-for-text "Saved"
neroli browser network --url-pattern /api/ --limit 20
```
